# Auto Folder Creator

A simple Python script that uses the OS module to create folders and files automatically.

## Description

This repository contains a Python script that demonstrates how to use the `os` module to create directories and files programmatically. It's useful for automating folder and file creation tasks.

## Features

- Creates a new folder.
- Creates files inside the created folder.
- Easy to customize for your needs.
